# 🎮 跳跃小鸟 (Flappy Bird) - Python 版

蓝色小鸟在前进，躲避绿色管道（类似马里奥管道）的小游戏！

## 🚀 快速开始

### 1. 安装依赖
```bash
pip install pygame
```

### 2. 运行游戏
```bash
python flappy_bird.py
```

## 🎯 游戏操作

| 操作 | 按键 |
|------|------|
| 跳跃/开始 | 空格键 |
| 跳跃/开始 | 鼠标点击 |
| 重新开始 | 空格键 / 鼠标点击 |
| 退出 | ESC 键 |

## 🎨 游戏特色

- 🐦 **蓝色小鸟** - 带有翅膀动画和旋转效果
- 🟢 **绿色管道** - 马里奥风格的管道
- ☁️ **动态云朵** - 天空背景飘动的云
- 📊 **实时计分** - 通过管道自动计分
- 💥 **碰撞检测** - 碰到管道或边界游戏结束

## 🛠 技术栈

- **Python 3.13**
- **Pygame 2.6.1** - 游戏开发框架
